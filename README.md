# digital-weighing-scale-load-cell
A complete digital weighing scale system using a strain gauge load cell. It covers theoretical analysis, LTspice circuit simulation, real-time signal testing via Analog Discovery (WaveForms), and an Arduino-based calibration and digital display readout.
